# Release Notes - Version 26.2630.4057
Please see the associated GitHub Issues for more information on any of these entries.


## Add HeaderCodes to MachineConfig section
The HeaderCodes string array has been added to the MachineConfig section of the batch file.

## Include Safe Height comment tool setup
Examples have been added to Docs/CommandLines.md showing one way to specify the SafeHeight: {Value} setting in the file.

## Add ReportGCodeOverlap action
/action:ReportGCodeOverlap /infile:{Filename} is now supported. See README.md for information.

## Add OptimizeGCode action
/action:OptimizeGCode /infile:{Filename} /outfile:{Filename} is now supported. See README.md for information.
